作用域链（Scope Chain）：变量查找的寻宝地图

Touko2021-03-162021-04-03

作用域链是 JavaScript 中变量查找的核心机制，它就像一张寻宝地图，指引引擎在代码中找到你需要的变量！

一、作用域链的本质：层层嵌套的查找系统

1.1 核心概念

作用域链是一种变量查找机制，当代码访问一个变量时，JavaScript 引擎会按照特定的顺序在多个作用域中查找该变量。这个查找顺序形成了一条”链”。

  graph LR
    A[当前作用域] --> B[父级作用域]
    B --> C[祖父级作用域]
    C --> D[...]
    D --> E[全局作用域]

1.2 创建时机

作用域链在函数定义时就已经确定，而不是在函数调用时！这点是理解闭包（👉 闭包（Closure）：JavaScript 的时空胶囊）的关键。

作用域链在函数创建时就已经确定，而不是在调用时！
