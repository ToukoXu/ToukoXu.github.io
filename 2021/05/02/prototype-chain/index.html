原型链是 JavaScript 面向对象编程的核心，它让对象能够继承属性和方法。

一、原型链：对象继承的 DNA

1.1 什么是原型链？

原型链是 JavaScript 实现继承的核心机制。每个对象都有一个隐藏的 [[Prototype]] 属性（可通过 __proto__ 访问），指向它的”原型对象”。当访问对象的属性时，如果当前对象没有该属性，就会沿着原型链向上查找。

1.2 原型链三要素

  1. prototype 属性：函数特有的属性，包含共享方法和属性
  2. __proto__ 属性（已弃用）：对象实例指向其构造函数的 prototype
  3. constructor 属性：指向创建该对象的构造函数

    B --> C[Object.prototype]

当你使用 new 关键字时，背后发生了四件事：

  1. 创建一个空对象 {}
  2. 设置该对象的原型为构造函数的 prototype
  3. 执行构造函数，this 指向新对象
