一、揭秘迭代器（Iterator）：数据遍历的魔法棒

1.1 迭代器是什么？

迭代器是 JavaScript 中的一种标准化协议（Iterator Protocol），它定义了如何按顺序访问集合中的元素。任何实现了迭代器协议的对象都可以称为迭代器。想象你有一本书📖，迭代器就像书签🔖，帮你记录阅读位置并获取下一页内容。

1.2 迭代器三要素

  1. [Symbol.iterator] 方法：返回迭代器对象
  2. next() 方法：返回 { value, done } 对象
    • value：当前迭代的值
    • done：布尔值，表示迭代是否完成
  3. 迭代状态：内部记录当前位置

1.3 可迭代对象

只要实现了 [Symbol.iterator]() 方法，它就是可迭代对象，它们可以被以下结构消费：

  • for...of 循环
  • 展开运算符 (...)
  • 解构赋值
  • Array.from()
  • Promise.all(), Promise.race()
  • Map, Set, WeakMap, WeakSet 构造函数

二、解构：数据拆箱的神奇钥匙

2.1 解构的本质

解构赋值（Destructuring）是基于迭代器协议实现的语法糖。当对数组或可迭代对象进行解构时，JavaScript 引擎会调用对象的 [Symbol.iterator]() 方法获取迭代器，然后按顺序调用 next() 方法获取值。
